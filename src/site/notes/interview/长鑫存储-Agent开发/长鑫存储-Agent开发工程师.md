## 📋 职位详情

**核心职责：**
1. 优化Agent应用性能效率和产出效果，深度理解Agent的实现机制和业务流程，针对评测过程中发现的问题，结合具体案例进行根源分析，量化输出可落地的优化方向与改进建议

**任职要求：**
1. 掌握至少一种主流编程语言，如Python、Java、Go等，熟悉Web开发框架，能够高效地进行前后端开发工作，具备良好的数据结构和算法基础
2. 熟悉AI Agent相关技术，了解工具开发、Agent流程设计、评估系统、上下文工程等体系
3. 具有较强的学习能力和求知欲，能够快速掌握新知识、新技术，关注AI领域的发展动态，推动Agent应用的持续优化

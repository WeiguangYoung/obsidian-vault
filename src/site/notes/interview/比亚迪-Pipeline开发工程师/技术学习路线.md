## 📚 学习路线

### 🔥 第一优先级（必须掌握）

#### 1. Python 深度开发
- **基础扎实**
  - 数据类型、列表/字典推导式
  - 面向对象（继承、多态、Mixin）
  - 装饰器、生成器、上下文管理器
  - 类型提示（Typing）与 Pydantic
- **并发编程**
  - `asyncio`：事件循环、协程、Task、Future
  - `aiohttp` / `httpx` 异步 HTTP
  - `threading` / `concurrent.futures`
  - `multiprocessing`：进程池与共享内存
  - GIL 的理解与绕过策略
- **框架设计能力**
  - 插件架构（Plugin / Hook 模式）
  - 回调机制、事件驱动
  - 配置驱动开发（YAML/JSON Schema）
- **测试**
  - pytest：fixture、parametrize、mock
  - 异步测试（pytest-asyncio）

#### 2. Kubernetes 深度掌握
- **核心资源**
  - Pod / Deployment / StatefulSet / DaemonSet
  - Service（ClusterIP / NodePort / LoadBalancer）
  - ConfigMap / Secret / Ingress
  - PV / PVC / StorageClass 存储抽象
- **工作负载调度**
  - 节点亲和性（nodeAffinity）
  - Pod 亲和性与反亲和性
  - 污点（Taint）与容忍（Toleration）
  - 资源配额（ResourceQuota / LimitRange）
- **Helm Charts**
  - Chart 结构：Chart.yaml / values.yaml / templates
  - 模板函数与流水线
  - 依赖管理与子 Chart
- **Operator 模式**（了解）
  - CRD（Custom Resource Definition）
  - Controller / Reconciler 循环

#### 3. Argo Workflows
- **基础概念**
  - Workflow / Template / DAG / Steps
  - 入口点（entrypoint）与参数传递
  - 制品管理（Artifact：S3 / OSS）
- **进阶特性**
  - 并行步骤、循环（withItems / withParam）
  - 条件分支（when / continueOn）
  - 递归与暂停步骤
  - 输出参数与全局参数
- **事件驱动**
  - Argo Events：事件源（Webhook / S3 / Git）
  - Sensor：事件触发 Workflow
  - EventBus 配置
- **集成**
  - Argo CLI + API
  - K8s CronWorkflow 定时调度
  - WorkflowTemplate 复用

#### 4. Apache Airflow
- **DAG 开发**
  - DAG 定义：default_args、schedule、catchup
  - PythonOperator / BashOperator
  - KubernetesPodOperator（在 K8s Pod 中运行）
- **任务编排**
  - 依赖设置（`>>` / `<<` / `set_upstream`）
  - 触发规则（trigger_rule：all_success / all_done / one_failed）
  - 分支任务（BranchPythonOperator）
  - 动态 DAG 生成
- **执行器与调度**
  - SequentialExecutor / LocalExecutor
  - CeleryExecutor（分布式）
  - KubernetesExecutor（动态 Pod）
  - 调度器配置：dag_dir_list_interval、max_active_runs
- **监控告警**
  - SLAs（服务等级协议）
  - 回调函数：on_success_callback / on_failure_callback
  - 邮件告警
  - Prometheus 指标暴露

#### 5. Docker & CI/CD
- **Docker**
  - Dockerfile 最佳实践（多阶段构建）
  - docker-compose 本地开发环境
  - 镜像仓库（Harbor / Docker Hub）
- **CI/CD 流水线**
  - GitLab CI：`.gitlab-ci.yml` 多阶段 Pipeline
  - Jenkins Pipeline（声明式/脚本式）
  - ArgoCD：GitOps 部署模式
  - 构建-测试-部署全流程

---

### ✅ 第二优先级（加分项）

#### 6. 数据存储技术
- **SQL**
  - 复杂查询（JOIN / 子查询 / CTE）
  - 窗口函数（ROW_NUMBER / RANK / LAG）
  - 索引优化、执行计划分析
  - 常用：PostgreSQL / MySQL
- **NoSQL**
  - Redis：缓存、队列、Pub/Sub
  - MongoDB：文档存储、聚合管道
- **消息队列**
  - Kafka：生产/消费、Topic/Partition、流处理
  - RabbitMQ：交换机/队列、任务分发
- **ETL 流程设计**
  - 数据抽取 → 清洗 → 转换 → 加载
  - 增量 vs 全量同步
  - 数据质量校验

#### 7. Git & GitLab
- 多仓库管理策略
- 分支保护、Merge Request
- CI/CD 多项目流水线（Multi-Project Pipeline）

#### 8. Linux 基础
- 常用命令、文件系统
- SSH 配置、密钥管理
- bash 脚本编写

---

### 🌟 第三优先级（提升竞争力）

#### 9. DevSecOps
- 开源合规扫描（FOSSA / Snyk）
- SBOM 生成（CycloneDX）
- 容器安全扫描（Trivy / Clair）
- 密钥管理（Vault / Sealed Secrets）

#### 10. 可观测性
- **监控**：Prometheus + Grafana
- **日志**：ELK / Loki
- **告警**：Alertmanager / 企微/钉钉通知

#### 11. IaC
- Terraform：基础设施声明式管理
- Ansible：配置管理与应用部署

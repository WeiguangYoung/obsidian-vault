# 🚗 奇瑞汽车 DevOps开发工程师 — 技术学习路线

## 1. DevOps 平台工具链
- GitLab / GitHub 代码托管与权限管理
- Jenkins Pipeline / GitLab CI 流水线编排
- Jira + Confluence 项目管理与文档沉淀
- Nexus / JFrog Artifactory 制品仓库管理
- Harbor 容器镜像仓库部署与维护
- SVN 迁移到 Git 的实操经验

## 2. 构建工具深度
- CMake 大型项目管理：子目录、跨平台编译
- Bazel 构建系统入门与迁移
- GTest 单元测试框架集成
- Cppcheck 静态分析规则定制

## 3. 容器与编排（核心）
- Docker：多阶段构建、Docker Compose、镜像优化
- Kubernetes：Pod / Deployment / Service / Ingress / ConfigMap / Secret
- Helm Chart 编写与发布
- K8s 集群运维：节点管理、资源配额、RBAC、网络策略
- 容器化 CI/CD 流水线设计

## 4. 可观测性体系
- Prometheus：指标采集、PromQL 查询、AlertManager 告警
- Grafana：Dashboard 搭建、面板设计、告警规则
- ELK / Loki 日志收集与检索
- 分布式链路追踪基础（Jaeger / OpenTelemetry）

## 5. 脚本与自动化开发
- Python：REST API 调用、自动化运维脚本、CI/CD 工具开发
- Shell：系统巡检、日志分析、批量操作
- 自研 DevOps 平台的设计思路与实现

## 6. 系统稳定性
- 故障排查方法论：构建失败、部署异常、网络问题
- 容量规划与资源优化
- 变更管理与回滚策略
- SLO / SLA / SLI 指标体系

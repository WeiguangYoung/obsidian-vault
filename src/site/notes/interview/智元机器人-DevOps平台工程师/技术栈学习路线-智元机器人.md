# 技术栈学习路线 — 智元机器人（DevOps 平台工程师方向）

> 🏢 **智元机器人 (Agibot)** — 平台工程师（DevOps）
> 定位：面向具身智能（人形机器人）的云端基础设施与 CI/CD 平台建设岗位
>
> 💡 核心差异：与传统 DevOps 不同，智元还涉及「多集群管理」「AI 提效工具链」「机器人仿真/数据处理平台」

## 📚 学习路线

### 🔥 第一优先级（必须掌握）

#### 1. Kubernetes 深度掌握（多集群管理）
- **核心资源精通**
  - Pod / Deployment / StatefulSet / DaemonSet
  - Service（ClusterIP / NodePort / LoadBalancer）
  - ConfigMap / Secret / Ingress / IngressController
  - PV / PVC / StorageClass / CSI
  - HPA / VPA / ClusterAutoscaler
- **多集群管理**
  - **KubeFed**（Kubernetes Federation）：跨集群资源分发
  - **Cilium ClusterMesh**：跨集群网络 & 服务发现
  - **Istio Multi-Cluster**：多集群服务网格
  - **kubectl ctx/ns** 多集群切换
  - **ArgoCD ApplicationSet**：多集群 GitOps 部署
  - 多集群监控聚合：Thanos / Grafana Mimir
- **集群运维**
  - 节点运维：Cordon / Drain / Taint / Tolerations
  - etcd 备份与恢复
  - 集群升级策略（kubeadm / EKS / ACK）
  - 网络插件对比：Calico / Cilium / Flannel
- **安全**
  - RBAC（Role / ClusterRole / Binding）
  - PodSecurityPolicy / OPA Gatekeeper / Kyverno
  - NetworkPolicy（零信任网络）

#### 2. CI/CD 流水线全栈
- **GitLab CI**
  - `.gitlab-ci.yml` 多阶段 Pipeline
  - 多项目 Pipeline（Multi-Project / Parent-Child）
  - 缓存（cache）与制品（artifacts）
  - Runner 类型：Shared / Group / Specific
- **Jenkins**
  - 声明式 Pipeline + Shared Libraries
  - 动态 Agent（K8s Plugin）
  - 多分支 Pipeline（Branch Source）
- **ArgoCD（GitOps）**
  - Application / ApplicationSet
  - Sync Policy（自动/手动/Prune）
  - Image Updater（自动镜像更新）
- **CI/CD 流程设计**
  - 代码提交 → 构建 → 单元测试 → 集成测试 → 部署
  - 灰度发布（Canary / Blue-Green）
  - 回滚策略（Rollback / Rollout Undo）
  - 多环境管理（Dev / Staging / Prod）

#### 3. 容器 & 镜像管理
- **Dockerfile 最佳实践**
  - 多阶段构建
  - 层优化（减少层数、合并 RUN）
  - 非 root 用户运行
  - 安全扫描（Trivy / Dockle）
- **镜像仓库**
  - Harbor：项目管理、复制、扫描
  - ECR / ACR / CR（云服务镜像仓库）
- **容器运行时**
  - Docker / containerd / CRI-O
  - 容器安全（Seccomp / AppArmor / Capabilities）

#### 4. Python 开发能力
- **基础扎实**
  - 数据类型、面向对象
  - 装饰器、生成器、上下文管理器
  - 异步编程（asyncio / aiohttp）
- **工程化**
  - FastAPI / Flask 后端开发
  - Pydantic 数据校验
  - pytest 测试
- **DevOps 工具开发**
  - Kubernetes Python Client（PyYAML + kubernetes SDK）
  - 云 API 封装（阿里云 SDK / AWS SDK）
  - CLI 工具开发（Click / Typer / Rich）
- **AI 提效**
  - 利用 AI Coding 工具（Copilot / Cursor / Claude）
  - 编写 AI Prompt 加速开发
  - 自动化代码审查脚本

#### 5. 可观测性
- **监控**
  - **Prometheus**：指标采集、PromQL 查询、Recording Rules
  - 自定义 Exporter 开发（Python / Go）
  - ServiceMonitor / PodMonitor（Prometheus Operator）
  - **Grafana**：仪表盘、告警规则、通知渠道
  - **Thanos**：多集群 Prometheus 聚合
- **日志**
  - **Loki**：日志采集（Promtail / Alloy）
  - **ELK**：Elasticsearch + Logstash + Kibana
  - 日志结构化（JSON 格式、日志级别）
- **链路追踪**
  - **Jaeger / Tempo**：分布式追踪
  - OpenTelemetry 标准
- **告警**
  - Alertmanager：分组、抑制、静默
  - 告警通知：企微 / 钉钉 / Slack / PagerDuty

#### 6. 云服务（阿里云 / AWS / 火山云）
- **阿里云**
  - ACK（容器服务 Kubernetes）
  - CR（容器镜像服务）
  - SLB / ALB / NLB 负载均衡
  - RDS / Redis / MongoDB
  - OSS 对象存储
  - SLS 日志服务
  - RAM 权限管理
- **AWS**
  - EKS（Elastic Kubernetes Service）
  - ECR（Elastic Container Registry）
  - EC2 / VPC / S3 / RDS
  - CloudWatch 监控日志
  - IAM 身份与访问管理
- **多云管理**
  - 跨云网络互通（VPN / CEN / PrivateLink）
  - 多云 CI/CD 策略
  - 成本管理（Cost Explorer / 预算告警）

### ✅ 第二优先级（加分项）

#### 7. Go 语言
- 基础语法、并发（goroutine / channel）
- Kubernetes Operator 开发（kubebuilder）
- CLI 工具开发（Cobra / Viper）
- gRPC 服务开发
- Prometheus Exporter 开发

#### 8. Linux 系统管理
- 系统调优（内核参数、文件描述符）
- 网络排查（tcpdump / netstat / ss / iptables）
- 磁盘管理（LVM / 文件系统扩展）
- systemd 服务管理

### 🌟 第三优先级（提升竞争力）

#### 9. 机器人平台基础（了解即可）
- **仿真平台**
  - Isaac Sim / Gazebo / MuJoCo
  - 仿真数据生成与回放
- **数据处理**
  - 传感器数据（相机/激光雷达/IMU）
  - 数据标注与回放工具
  - 数据管道设计
- **工具链开发**
  - 可视化工具（Foxglove / RViz）
  - 自动化测试平台
  - 数据集管理平台

#### 10. 基础设施即代码（IaC）
- **Terraform**
  - HCL 语法、Provider 管理
  - 模块化设计（Module）
  - 状态管理（Remote State / Lock）
- **Ansible**
  - Playbook / Role 编写
  - 配置管理与应用部署

#### 11. 服务网格（Service Mesh）
- Istio 核心概念（VirtualService / DestinationRule）
- 流量管理（灰度发布 / 故障注入）
- 安全（mTLS / AuthorizationPolicy）

# 🤖 智元机器人 DevOps平台工程师 — 技术学习路线

## 1. Kubernetes 深度（核心）
- K8s 核心资源：Pod / Deployment / StatefulSet / DaemonSet / Job / CronJob
- 网络：CNI / Service / Ingress / NetworkPolicy
- 存储：PV / PVC / StorageClass / CSI
- 调度：亲和性/反亲和性、污点与容忍、资源配额
- 安全：RBAC、PodSecurityPolicy、NetworkPolicy
- Helm：Chart 开发、Repo 管理、Release 生命周期

## 2. 多集群管理
- 海内外多集群架构设计
- 集群联邦（KubeFed）基础
- 跨集群发布策略：灰度、金丝雀、蓝绿
- 多集群统一监控与告警

## 3. CI/CD 流程改造
- 统一 CI/CD 平台设计与落地
- GitOps 实践：ArgoCD / Flux CD
- 多环境流水线：开发 → 测试 → 预发 → 生产
- 容器镜像构建加速与安全扫描

## 4. 云平台实战
- 阿里云：ACK（容器服务）、SLB、OSS、NAS
- AWS：EKS、ECR、CloudWatch、Route53
- 火山引擎：VKE 容器服务
- 多云架构设计：成本优化、灾备、合规

## 5. 可观测性
- 监控：Prometheus + Grafana 全栈
- 日志：ELK / Loki / 阿里云 SLS
- 链路追踪：Jaeger / SkyWalking
- 告警治理：分级、收敛、值班轮转

## 6. 机器人工具链开发
- 仿真平台部署与运维
- 数据处理流水线搭建
- 可视化工具集成
- 自动化测试平台对接

## 7. 编程能力
- Python：Web 框架(FastAPI/Flask)、SDK 开发、自动化脚本
- Go：K8s client-go、Operator 开发、CLI 工具
- AI Coding 工具：Copilot / Cursor 提升开发效率

## 8. SRE 方法论
- SLO / SLI / Error Budget
- 故障演练与混沌工程（Chaos Mesh）
- On-Call 与事件响应流程
- 事后复盘与改进闭环

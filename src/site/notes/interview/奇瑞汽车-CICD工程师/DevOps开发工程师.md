# 🚗 奇瑞汽车 — DevOps开发工程师（岗位JD）

> 📅 2026-07-16 获取

---

## 岗位职责

1. 负责DevOps平台软件研发工作，搭建CI/CD自动化工具链，提高软件项目开发运维效率及质量。
2. 负责产品线CI/CD持续集成交付工作，解决项目DevOps相关的各种技术问题。
3. 部署和维护DevOps研发系统，包括 **Jira / Git / CI/CD / Jenkins / SVN / Harbor / Nexus** 等，编写相关技术文档，开展DevOps相关培训工作。
4. 设计、开发企业自有DevOps基础设施，解决项目DevOps相关的基础架构问题。
5. 熟悉 **K8s 和 Docker** 容器化相关技术栈，能够处理K8s和容器相关问题。
6. 设计监控系统，实时监控系统健康状况，负责系统稳定性保障，快速响应并解决潜在故障。

## 任职要求

1. 熟悉DevOps相关工具和技术，如 **Jira / Git / CI/CD / Jenkins / SVN / Harbor / Nexus / Jfrog** 等。
2. 熟悉常用的构建工具，如 **CMake / Bazel**。
3. 熟悉 **GTest** 等C/C++单元测试框架，掌握 **Cppcheck** 等静态代码分析工具，具备性能分析和优化经验。
4. 具备丰富的 **Docker / K8s** 容器化运维实践经验，熟悉云原生应用。
5. 能熟练使用 **Python / Shell** 等编程语言进行自动化工具开发。
6. 熟悉 **Grafana、Prometheus** 等监控系统的部署和运维。
7. 具备系统化解决问题的能力，良好的沟通能力、抗压力和自驱力强。

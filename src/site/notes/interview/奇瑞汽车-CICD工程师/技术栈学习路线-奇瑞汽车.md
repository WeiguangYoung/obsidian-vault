## 📚 学习路线

### 🔥 第一优先级（必须掌握）

#### 1. Linux 系统管理
- 文件系统、权限管理、进程管理
- 网络配置（网卡、路由、iptables）
- systemd 服务管理、cron 定时任务
- 常用命令：grep / awk / sed / find / xargs
- 系统调优：文件描述符、ulimit、内核参数

#### 2. Git & GitLab 代码管理
- 分支策略：Git Flow / Trunk Based Development
- Merge Request / Code Review 流程规范
- GitLab CI 流水线配置（`.gitlab-ci.yml` 编写）
- 多仓库管理 & Submodule / Subtree
- 版本发布 Tag 管理与 Changelog

#### 3. CI/CD 深度 — Jenkins Pipeline
- **声明式 Pipeline**（Declarative Pipeline）
  - `pipeline { agent, stages, post }`
  - 环境变量、参数、选项
  - when 条件判断、并行执行
- **脚本式 Pipeline**（Scripted Pipeline）
  - Groovy 语法基础
  - Shared Libraries 封装与复用
- **插件生态**
  - SonarQube 扫描插件
  - Artifactory / Nexus 制品管理
  - Kubernetes Agent 动态节点
  - Blue Ocean 可视化
- **多节点流水线**
  - Master + Agent 架构
  - Docker / K8s 动态 Slave
  - Label 与节点选择

#### 4. Docker 容器化
- Dockerfile 编写（多阶段构建、层优化）
- docker-compose 多服务编排
- 镜像仓库管理（Harbor / Nexus）
- 容器作为编译环境（SDK 镜像、工具链镜像）
- 构建缓存、层压缩、安全扫描（Trivy）

#### 5. 嵌入式构建流程
- **Make / CMake**
  - CMakeLists.txt 编写（target / library / executable）
  - 工具链文件（toolchain.cmake）
  - 条件编译、安装规则
- **交叉编译**
  - ARM GCC 工具链配置
  - --sysroot、动态库/静态库
  - 编译选项：-O 优化级别、-Wall、-fPIC
- **构建环境管理**
  - License 服务器（FlexNet / 浮动 License）
  - 依赖库版本锁定
  - 构建一致性（Docker 化 + 版本锁）

#### 6. 质量门禁工具链
- **SonarQube**
  - 规则配置（Quality Gate）
  - 扫描集成（sonar-scanner）
  - 门禁阻断机制
- **静态代码分析**
  - MISRA C/C++ 规范检查
  - Coverity / Polyspace 深度分析
  - QAC 代码审查
- **单元测试**
  - Google Test（C++）
  - CUnit（C）
  - CMock（Mock 框架）
- **覆盖率统计**
  - gcov / lcov
  - 覆盖率门禁（行覆盖 + 分支覆盖）
- **开源合规**
  - FOSSA / Black Duck
  - SBOM（CycloneDX / SPDX）

### 🌟 第三优先级（入职后学习）

#### 10. 汽车行业知识
- **ASPICE**：软件工程流程等级
- **ISO 26262**：功能安全（ASIL A-D 等级）
- **ISO/SAE 21434**：信息安全
- **AUTOSAR CP / AP**：经典平台 / 自适应平台
- **需求-代码-测试追溯**：DOORS / Polarion / Jama

#### 11. 测试环境对接
- **HIL / SIL**：硬件在环 / 软件在环
- **CANoe**：CAN / LIN / Ethernet 仿真
- **vTESTstudio**：自动化测试用例开发
- **dSPACE / ETAS / INCA**：ECU 标定与测量
- **UDS 诊断 / 刷写协议**
- **Vector Flash Bootloader**
